# webpack-v3-sandbox
Try webpack v3


# Setup and lunch web server
```
$ git clone git@github.com:tayutaedomo/webpack-v3-sandbox.git
$ cd webpack-v3-sandbox
$ npm install
$ bin/www
```

Your app should now be running on [localhost:3000](http://localhost:3000).


# Development
## Using Watch Mode
```
$ npm run watch
```


# Trials
